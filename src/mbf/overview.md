# `MakeBlendField`

::: warning
Many parts of `MakeBlendField` remain a mystery, so this part of the documentation is (largely) incomplete.
:::

zombs.io employs an original method to validate players via PoW challenges that the client has to solve in order to get in a server and maintain connection. Internally, it is called `MakeBlendField`. From now on, for convenience, `MakeBlendField` will simply be referred to as MBF.

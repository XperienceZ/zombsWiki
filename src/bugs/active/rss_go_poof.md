# Disappearing Resources

## Phenomenon

Occasionally, Trees and Stones (and likely also Neutral Camps, but no such cases have been reported for them) previously found can disappear after some time in a server.

## Cause

Unknown.

## Examples

### April 2026

On 26th April, a discord user named Shadow sent a screenshot showing him sitting on the missing rock's place. The rock with the UID 657 is completely gone.

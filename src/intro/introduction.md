# What is zombs.io?

::: warning
This section of the documentation is incomplete.
:::

[zombs.io](https://zombs.io/) is a 2D multiplayer survival video game, developed by an American studio - End Game, that centers around building bases to defend against zombies at night. It was released in 2017 on the web. PvP factors are also available, and players can join parties with up to 3 other people.

## Gameplay

There is a simple tutorial that teaches the basics of the game with a series of hints.
